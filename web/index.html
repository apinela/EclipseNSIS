EclipseNSIS

EclipseNSIS is a plugin for the Eclipse platform which enables editing, compiling and testing Nullsoft Installer (NSIS) scripts.

Features

  • Syntax highlighting editor.
  • Outline view for quick navigation.
  • Integrated compiling and testing support.
  • Error and warning marker generation.
  • Script wizard.
  • Code templates.
  • Integrated NSIS help.

Screenshot

Check out the User Guide for more screenshots.

Download

There are two ways to download EclipseNSIS:

  1. (Recommended) Use the Eclipse Software Updates utility to connect to http://eclipsensis.sf.net/update and download the desired release.
  2. Download directly from SourceForge and unzip into the Eclipse folder.
