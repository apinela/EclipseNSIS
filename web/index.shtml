EclipseNSIS

EclipseNSIS is a plugin for the Eclipse platform which enables editing, compiling and testing Nullsoft Installer (NSIS) scripts. The EclipseNSIS InstallOptions Editor plugin is an add-on to EclipseNSIS which enables editing of InstallOptions scripts.

Features

EclipseNSIS

  • Requires Eclipse 3.5 or better.
  • Requires Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003, Windows Vista, or Windows 7.
  • Syntax highlighting editor.
  • Outline view for quick navigation.
  • Integrated compiling and testing support.
  • Error and warning marker generation.
  • Script wizard with template support.
  • Command wizard.
  • Code templates.
  • Integrated NSIS help.
  • NSIS Update Wizard.

EclipseNSIS InstallOptions Editor

  • Requires Eclipse 3.5 or better, EclipseNSIS and Eclipse GEF 3.5.
  • Requires Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003, Windows Vista, or Windows 7.
  • Design editor with point & click editing and outline view.
  • Source editor with syntax highlighting, code outlining & folding, dynamic error checking and quickfix.
  • Templates.
  • Integrated help.
  • InstallOptions preview.
  • Script wizard with template support.

Download

There are two ways to download EclipseNSIS plugins:

  1. (Recommended) Use the Eclipse Software Updates and Add-ons utility or the classic Eclipse Update Manager to connect to http://eclipsensis.sf.net/update and download the desired release.
  2. Download directly from SourceForge and unzip into the Eclipse folder. However, using this method will NOT ensure that all prerequisites for using the plugins have been satisfied.
