EclipseNSIS is a plugin for the Eclipse platform which enables editing,
compiling and testing Nullsoft Installer (NSIS) scripts.

DIRECTORY STRUCTURE
===================
CreateEnv - Script for generating native build environment.
MakeNSISRunner - Sources for native code DLL for running MakeNSIS.exe.
net.sf.eclipsensis - EclipseNSIS plugin sources.
net.sf.eclipsensis.feature - EclipseNSIS feature sources.
net.sf.eclipsensis.site - EclipseNSIS update site sources.
web - Project web site (http://eclipsensis.sf.net) sources.
WinAPI - Sources for native code DLL for calling miscellaneous Windows API.
