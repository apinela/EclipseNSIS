EclipseNSIS Utilities Information

EclipseNSIS Utilities consists of miscellaneous Eclipse plugins developed for the EclipseNSIS project.
The utilities currently provided are:

  • JNIGen - A plugin for generating JNI header files from Java source or class files.
  • JARSigner - A plugin for digitally signing JAR files.
