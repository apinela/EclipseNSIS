NSIS Update

NSIS Update is a feature of EclipseNSIS that can be used to find, download and install the latest NSIS software. It can be invoked in several ways:

  • Menuitem/Toolbar button
  • Wizard
  • Scheduler

Please note the following items:

  1. NSIS Update uses the same HTTP proxy settings as the Eclipse platform Install/Update feature.
  2. NSIS Update scheduler does not update EclipseNSIS itself. To schedule automatic updates of EclipseNSIS (and other installed features in Eclipse) use the Eclipse platform Automatic Update Scheduler.
  3. NSIS Update does not update CVS or nightly builds of NSIS. If the configured NSIS home points to a CVS or nightly build, then it must be manually updated by recompiling it or downloading a new nightly build.

Menuitem/Toolbar Button

NSIS Update is available as menuitem in the EclipseNSIS menu and a button on the EclipseNSIS toolbar, displayed whenever the EclipseNSIS editor is open. This provides convenient, single-click access to NSIS Update. The action performed is equivalent to the default update action in the NSIS Update preferences page.

Wizard

The NSIS Update Wizard is available from the Help→NSIS Update Wizard menuitem. The user can specify the NSIS Update action and other update options before it runs. The NSIS Update action and options are identical to those described in the NSIS Update preference page.

Scheduler

NSIS Update is able to search for updates to NSIS on a periodic basis without user assistance. The way this search is performed is configurable by the user on the Automatic NSIS Updates preference page. Users can configure update schedule, actions and other options.
