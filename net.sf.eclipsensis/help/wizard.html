Introduction

The EclipseNSIS script wizard is a guide through the process of creating a new NSIS Installer script. The resulting script file is saved in the Eclipse Workspace and can be automatically compiled and tested if desired. The wizard uses only a subset of the features available in NSIS. For further information on script customization, please consult the NSIS Users Manual.

The script wizard can be accessed either through the EclipseNSIS editor menu or toolbar, or by accessing the Eclipse Workbench New button or menu  and selecting the Other... item. It consists of 6 pages, which are described in the following sections.

In each section, click on the images to get help about a particular script wizard item.
