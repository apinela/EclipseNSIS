Installation

System Requirements

  1. Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003, Windows Vista, or Windows 7.
  2. NSIS 2.0 or better. http://nsis.sourceforge.net
  3. Eclipse 3.5 (Galileo) or better.
  4. JRE 5.0 or better. EclipseNSIS has been tested with the following JREs:
    1. Sun

Getting EclipseNSIS

There are two ways to get and install EclipseNSIS:
  1. (Recommended) Use the Eclipse Software Updates and Add-ons utility or the classic Eclipse Update Manager to connect to http://eclipsensis.sf.net/update and download the latest release.
  2. Download directly from SourceForge and unzip into the Eclipse folder.
