Installation

Prerequisites

  1. Microsoft Windows 98, Windows ME, Windows 2000 or Windows XP.
  2. NSIS 2.0 or better. http://nsis.sourceforge.net
  3. Eclipse 3.0 or above.

Getting EclipseNSIS

There are two ways to get and install EclipseNSIS:
  1. (Recommended) Use the Eclipse Software Updates utility to connect to http://eclipsensis.sf.net/update and download the desired release.
  2. Download directly from SourceForge and unzip into the Eclipse plugins folder.
