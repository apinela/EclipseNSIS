Installation

System Requirements

  1. Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003 or Windows Vista.
  2. NSIS 2.0 or better. http://nsis.sourceforge.net
  3. Eclipse 3.3 or better.
  4. JRE 1.4.1 or better. EclipseNSIS has been tested with the following JREs:
    1. Sun
    2. IBM
    3. BEA JRockit

Getting EclipseNSIS

There are two ways to get and install EclipseNSIS:
  1. (Recommended) Use the Eclipse Update Manager to connect to http://eclipsensis.sf.net/update and download the desired release.
  2. Download directly from SourceForge and unzip into the Eclipse folder.
