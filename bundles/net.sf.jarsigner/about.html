JARSigner

JARSigner is an Eclipse plugin for digitally signing JAR files and verifying digitally signed JAR files.

It is implemented in 100% Java and satisfies the JAR signing Specification.
To install or update, visit EclipseNSIS Update Site in the Update Manager.

Usage:

Right-click on one or more .jar or .zip files in any of the Eclipse navigator views and select the menu item named "Sign JAR" or "Verify Jar". Follow the instructions in the dialog which is displayed. If the dialog cannot automatically determine the location of the Java SDK tools.jar, you will have to manually find it in the filesystem.
