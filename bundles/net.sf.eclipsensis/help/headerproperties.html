Header Properties

The EclipseNSIS Header Properties page is displayed for NSIS header files (*.nsh). It is used to associate an NSIS script file (*.nsi) with the header file..

Associated NSIS Script
The NSIS script that is associated with the header file. Use the browse button to select a script file from the Eclipse workspace.
