Installation

System Requirements

  1. EclipseNSIS 0.9.5
  2. Eclipse Graphical Editing Framework (GEF) 3.2 or better

Getting EclipseNSIS InstallOptions Editor

There are two ways to get and install EclipseNSIS InstallOptions Editor:
  1. (Recommended) Use the Eclipse Update Manager to connect to http://eclipsensis.sf.net/update and download the desired release.
  2. Download directly from SourceForge and unzip into the Eclipse folder.
