EclipseNSIS InstallOptions Editor User Guide

EclipseNSIS InstallOptions Editor is a plugin for the Eclipse platform which enables creating and editing InstallOptions scripts used by the Nullsoft Installer (NSIS).
To install or update, visit the EclipseNSIS Update Site in the Update Manager.
